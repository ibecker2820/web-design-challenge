Summary: Latitude vs.

The idea behind this project was to compare latitude to different aspects. We used information from OpenWeatherMap API to recieve data from over 500 cities.

We used Pandas to create the dataframe. From this dataframe we were able to use Matplot lib to plot different weather aspects against latitude. We looked at four aspects: temperature, humidity, clouidness, and wind speed.
